# Подсказка по GIT

Создание репозитория:
```sh
git init
```
Добавление содержимого католога:
```sh
git add
```
Команда сохранения:
```sh
git commit -m "Message"
```
Журнал изменений:
```sh
git log
```
Журнал изменений в сокращении:
```sh
git log --oneline
```
Переключение между версиями:
```sh
git checkout <имя ветки>
```
Отображение всех веток:
```sh
git branch
```
